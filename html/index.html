HTML-теги и форматирование текста

HTML теги - основа языка HTML. Каждый тег несет в себе определенную информацию, может описывать документ в общем или способ форматирования текста. Все содержимое веб-страницы задается с помощью тегов.

Тег помещается в угловые скобки <тег>. Чаще всего для тега задается парный закрывающий тег, который начинается с символа /, но в некоторых случаях он отсутствует, например в теге <br>, который устанавливает перевод строки.
Информация, заключенная между открывающим и закрывающим тегом, называется его контейнером.

С помощью HTML-тегов можно задавать индексы в описании формул. Например, формулу квадрата суммы: (a+b)2=a2+2ab+b2.
Или молекулярную формулу декана: C10H22.

Тег <pre> используется для вывода текста без форматирования. Это нужно потому, что все пробелы, переносы строк и табуляция в коде HTML-страницы заменятются на один пробел.
Этот    тег удобен для вывода листинга "Hello, world!" на C++. 

          #include <iostream>

          class World
          {
              public:
                  static int Hello()
                  {
                    std::cout << "Hello, world!" << std::endl;
                      return 0;
                  }
          };

          int main()
          {
              return World::Hello();
          }

Таблицы

Тег <table> служит контейнером для элементов, определяющих содержимое таблицы. Любая таблица состоит из строк и ячеек. Примером может служить таблица допустимых указаний типов документов в зависимости от версии языка.

Допустимые DTD
DOCTYPE Описание
HTML 4.01
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd"> Строгий синтаксис HTML.
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd"> Переходный синтаксис HTML.
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd"> В HTML-документе применяются фреймы.
HTML 5
<!DOCTYPE html> Для всех документов.
XHTML 1.0
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd"> Строгий синтаксис XHTML.
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd"> Переходный синтаксис XHTML.
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd"> Документ написан на XHTML и содержит фреймы.
XHTML 1.1
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd"> Разработчики XHTML 1.1 предполагают, что он постепенно вытеснит HTML. Никакого деления на виды это определение не имеет, синтаксис один и подчиняется четким правилам.
